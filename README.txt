||||| CONTROLS ||||||

Forward = W
Left = A
Back = S
Right = D
Up = E
Down = Q
Move Faster = Left Shift (Hold)

Move Obstacle Marker Forward = 1
Move Obstacle Marker Back = 2
Place Obstacle = 3

Rotate Camera = RMB (Hold)
Zoom In/Out = Mouse Scroll Wheel
